Bethune Computer Science Club Website

May 2021

This website was built for the Bethune Computer Science Club completely from scratch with the collaboration with my friends using React.js. I coded most of the frontend and all of the backend for this site. Originally, I chose to use the MERN Stack (MongoDB, Express, React and Node.js) but in the end, I chose to switch to Firebase after realizing that a server had to be hosted in order to communicate with the backend. Firebase hosting was used to host the site, Firebase storage and firestore was used to store the backend pictures and data and Firebase authentication was used to authenticate admins for the site. This was my first web development project and I had a lot of fun with it.

Garage Door Opener

April 2021

This project was built using an ESP 01 module, a microcontroller that has wifi capabilities. The ESP01 Module is wired to the garage door opener through a relay. When the user taps a button on the app, the app sends a code via MQTT to the ESP 01 Module, which then sends a one second pulse to the garage door opener. This then opens the garage door. The ESP 01 module must be connected to a home wifi network for the MQTT communication to work. However, the phone with the app just needs to be connected to the internet in any way including using cellular data or wifi.
